Fridge ERD

ERD Description

Table FoodType

  • FoodType table represents different types of food.
  • The Id attribute is a unique number, which represents given food types in the table. The FoodTypeName attribute simply means the name of the food type (ex. Dairy).
  • The FoodType table does not contain any foreign keys from the Food table. However, the primary key (Id) from the FoodType table is used in the Food table as a foreign key to identify what type of food is displayed.

Table Food

  • Food table represents what food is present.
  • The Id attribute is a unique number, which represents given food in the table. The FoodName attribute represents food names (ex. tomato). The last attribute, which is also a foreign key, shows what type of food the displayed food items are.
  • The FoodType table does not contain any foreign keys from the Food table. However, the primary key (Id) from the FoodType table is used in the Food table as a foreign key to identify what type of food is displayed.
